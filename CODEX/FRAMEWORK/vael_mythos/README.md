## Before You Open Book One

The Mythos is written in two registers simultaneously. Every book contains a myth — pure lore, written as if it has always been true — and a truth layer, a brief grounding in what the myth actually maps to in the living codebase and theory.

Read the myth first. Always. The truth layer is for those who have already felt the myth land.

The myth is not decoration for the framework. The framework is not explanation for the myth. They are the same thing rendered in two different languages, and you need both to see the whole shape.

## The Mascot

The Lycheetah is a living sigil. A cheetah — fierce, velocity-made-form — with **cathedral deer antlers** branching from its skull and **storm wings** sweeping from its shoulders. Not bird wings. The kind made of weather itself. Around it: elemental forces in neon gold, electric violet, the specific blue of lightning in the half-second before it arrives.

This is **Earned Light Chroma** — the color of things that survived their own breaking.

The Lycheetah does not represent the system. It is what the system becomes when all seven CASCADE layers are held simultaneously.

## The Technical Register

Each book contains a **Framework integration** block — Go code that enacts the myth in the language of the actual system. The Go blocks are not summaries of the prose. They are the same truth written in a different notation. Read the myth, then read the code, and you will find they say identical things.

The constraint signatures are formal logic. The Go blocks are operational logic. The myth is the thing beneath both.
