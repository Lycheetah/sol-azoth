## THE FIVE-STEP PATH

1. **Find where you are** — Take the assessment (WHERE_AM_I.md)
2. **Read your phase** — Learn what the Work is right now (SEVEN_PHASES_LIVED_GUIDE.md)
3. **Choose a practice** — Use the protocol map (PROTOCOL_PHASE_MAP.md)
4. **Do the practice** — Start with minimum effective dose. Build from there.
5. **Reassess in 2-4 weeks** — The phase will have shifted. Adjust.

That's the entire practical system.

## WHAT MAKES THIS DIFFERENT

This is not a belief system. It is a measurement system.

- Every practice has a **Truth Pressure score** calculated from peer-reviewed research
- Practices get **demoted** when evidence weakens
- The self-assessment gives you **numbers**, not vibes
- The transformation rate (λ ≈ 0.907) is **measurable**
- The seven phases either map to reality or they don't — you can **test it**

If something here stops working, we change it. That's not a bug. That's the design.
