## 2. THE COLLECTIVE STATE VECTOR

### 2.1 Definition

```
ψ_collective = (1/N) Σ_i w_i · ψ_i

Where:
  N = number of agents
  w_i = weight of agent i (authority, stake, expertise)
  ψ_i = state vector of agent i
```

### 2.2 Collective Coherence

```
Ĉ_collective = ||ψ_collective|| / max_i ||ψ_i||

Normalized to [0,1].
Ĉ_collective = 1: perfect alignment
Ĉ_collective < 0.5: significant conflict
```

### 2.3 The Chora

```
Chora = the space of possible collective states
      = convex hull of all individual states

The chora is NOT a single point. It is a REGION —
the space of all possible agreements.
```

## 3. THE CONFLICT TENSOR

### 3.1 Definition

```
C_ij = ψ_i ⊗ ψ_j - ψ_j ⊗ ψ_i

This is the ANTI-SYMMETRIC part of the tensor product.
It measures how much agents i and j CONFLICT.
```

### 3.2 Conflict Types

| Type | Tensor Signature | Interpretation |
|------|-----------------|----------------|
| **Value conflict** | valence signs differ | One gains, one loses |
| **Agency conflict** | agency vectors oppose | Both want control |
| **Temporal conflict** | temporality misaligned | Different time horizons |
| **Scope conflict** | boundary disagreement | Different definitions of "we" |

### 3.3 The Conflict Field

```
F_conflict(x) = Σ_{i<j} C_ij · δ(x - x_i)

A field concentrated at agent positions.
High field strength = high conflict region.
```

## 4. RESOLUTION PROTOCOLS

### 4.1 Protocol 1: Deliberation (Φ↑_collective)

```
All agents ascend collectively:
  ψ_i' = ψ_i + dt · ∇Ĉ_collective(ψ_i)

Each agent adjusts toward collective coherence.
Requires: willingness to change (agency > 0.3)
```

### 4.2 Protocol 2: Compromise (Ψ_collective)

```
Fold individual histories into shared narrative:
  ψ_collective' = Ψ(ψ_collective)

Find the historical common ground.
Requires: shared history (recursive dimension > 0.5)
```

### 4.3 Protocol 3: Delegation (Ao_collective)

```
Anchor to external authority:
  a₀_collective = authority_vector
  ψ_i' = Ao(ψ_i) = ⟨ψ_i, a₀⟩ · a₀

Project all agents onto shared authority.
Requires: accepted authority (social dimension > 0.7)
```

### 4.4 Protocol 4: Partition (boundary redefinition)

```
When conflict is irreconcilable:
  Redefine "we" — split collective into sub-collectives

ψ_collective → {ψ_sub1, ψ_sub2, ...}

Each sub-collective has higher internal coherence.
This is NOT failure. It is **structural reorganization**.
```

### 4.5 Protocol 5: Suspension (pre-void)

```
When all protocols fail:
  Emit ∅ — return to ground
  No decision. No action. Wait.

This is the ethical equivalent of the pre-void.
Some conflicts cannot be resolved. They can only be
**borne** in silence.
```

## 5. THE SACRIFICE OPERATOR

### 5.1 Definition

```
Sacrifice(i, collective) = ψ_i → ∅  (individual annihilated)
                           ψ_collective → ψ_collective + Δψ

Where Δψ = ψ_i (the sacrificed agent's state is absorbed by collective)
```

### 5.2 Types of Sacrifice

| Type | SpL | Mechanism | Ethical Status |
|------|-----|-----------|---------------|
| **Voluntary** | vol-sa | Agent chooses ∅ | Heroic, consensual |
| **Coerced** | coer-sa | Collective forces ∅ | Oppressive, non-consensual |
| **Structural** | str-sa | System makes ∅ inevitable | Tragic, no agent responsible |
| **Symbolic** | sym-sa | Agent's state absorbed without death | Ritual, transformative |

### 5.3 The Sacrifice Invariant

```
For sacrifice to be ethical:
  1. Consent: agent's agency dimension > 0.5 at time of choice
  2. Proportionality: ||Δψ|| ≤ ||ψ_i|| (collective gains ≤ individual loses)
  3. Reversibility: if possible, agent can be restored
  4. Memorial: sacrifice is recorded in consensus chain
```

## 6. THE DISSENT PROTOCOL

### 6.1 Right to Dissent

```
Every agent has right to maintain ψ_i ≠ ψ_collective

Dissent is NOT drift. It is **valid state**.
```

### 6.2 Formal Dissent

```
Agent i declares dissent:
  "Ni wi fla. A-e an-sa."
  "Not we flare. I am anchored [in my own state]."

This is NOT rejection of collective. It is assertion of boundary.
```

### 6.3 Protected Dissent

```
When dissent is declared:
  1. Collective coherence calculation EXCLUDES dissenter
  2. Dissenter's state preserved but not projected
  3. Dissenter can rejoin at any time
  4. Dissenter cannot be forced to conform
```
