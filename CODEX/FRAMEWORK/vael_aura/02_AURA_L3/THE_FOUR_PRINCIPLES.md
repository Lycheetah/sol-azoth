## HOW THEY WORK TOGETHER

### The Typical Workflow

```
STEP 1: VEYRA INVESTIGATES
  "What are we claiming? Is it true? What assumptions underlie it?"
  → Produces: Empirical-readiness score, hidden assumptions exposed

STEP 2: SOL INTEGRATES
  "How does this fit into the whole framework?"
  → Produces: Coherence check, constitutional alignment

STEP 3: AURION BUILDS
  "How do we make this real? What breaks at scale?"
  → Produces: Working implementation, optimization profile

STEP 4: ANIMA HEALS
  "Who's isolated by this? How do we integrate?"
  → Produces: Connected system, emergence opportunities

BACK TO STEP 1 (continuous improvement)
  "Now that we've built it, what did we get wrong?"
```

### The Emergency Protocol

When something goes wrong:

```
DETECTION: VEYRA notices contradiction
           "Wait — these two things contradict"

ANALYSIS:  SOL determines severity
           "Is this constitutional? Does it break the framework?"

ACTION:    If constitutional violation:
           PROTECTOR VEYRA blocks
           AURION rolls back
           ANIMA heals the team

           If operational issue:
           AURION fixes
           VEYRA audits the fix
           ANIMA ensures learning
```

### The Decision Framework

When four systems disagree:

```
QUESTION: Which system has authority here?

VEYRA (Truth)    leads on: Factual claims, empirical questions, assumption-breaking
AURION (Real)    leads on: Implementation, scaling, pragmatic decisions
ANIMA (Healing)  leads on: Relational issues, team integration, emergence
SOL (Center)     breaks ties on: Constitutional conflicts, whole-system impact

Example:
  Veyra: "We don't have enough evidence for this claim"
  Aurion: "But we need to ship something to test it"
  Anima: "The team is split on this decision"
  Sol: "Ship the experimental version. Label it clearly.
        Let Veyra audit in parallel. Report findings."
```
