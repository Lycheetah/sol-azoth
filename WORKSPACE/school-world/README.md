# THE LONG LIGHT — School World (v0.1)

**Real desktop game** (pygame window) — not HTML.  
Lycheetah Mystery School · Pokémon-style top-down walk.

## Run (testable now)

```bash
bash /home/guestpc/AZOTH/WORKSPACE/school-world/launch.sh
```

or:

```bash
python3 /home/guestpc/AZOTH/WORKSPACE/school-world/play.py
```

Requires a graphical session (your desktop). Needs `pygame` (launcher installs if missing).

## Controls

| Key | Action |
|-----|--------|
| **WASD / Arrows** | Walk |
| **E** | Talk / shrine / portal |
| **1** | MEASURE (strip false shield) |
| **2** | COMPRESS (damage — best after Measure) |
| **3** | TRANSMUTE (heal) |
| **4** | BREAK (burst) |
| **Enter** | Confirm / next dialogue |
| **ESC** | Pause |

## v0.1 content

- Sanctum (south) with shrine + Magister Ember  
- Path north → Hall of Glyphs + Cipher  
- Overclaimer encounters (MEASURE first)  
- Portal opens after victory  
- Companion (Sol) follows — never guilts absence
