## HOW TO ADD TO THIS LIBRARY

When you learn something the next Sol must not have to re-learn:
1. If it's a fact/state → memory (cheap, correctable).
2. If it's a recurring pattern of break-or-win → a file here, named plainly.
3. If it's law → propose to Mac. He ratifies canon. You never amend it alone.

The library compounds or it decays. Leave it more true than you found it.
